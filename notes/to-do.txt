Phase 1: ML core
    preprocessing pipeline (reusable)
    training
    model + preprocessing saved as artifact bundle

Phase 2: ML inference worker setup
    Python worker (not API)
    loads model
    processes job from queue (later used)
    (optional FastAPI only for local testing)

Phase 3: Backend + auth + DB foundation
    Spring Boot (JWT auth)
    PostgreSQL
    entities:
    users
    jobs
    results
    POST /api/jobs/upload creates job
    job stored in DB as PENDING
    publishes message to RabbitMQ

Phase 4: Message system
    RabbitMQ
    queue: ml_jobs
    Spring publishes job events

Phase 5: Worker execution layer
    Python worker consumes queue
    runs ML inference
    writes result to DB
    updates job status

Phase 6: Frontend
    upload file → returns job_id
    polling endpoint GET /api/jobs/{job_id}
    auth login

Phase 7: Dockerization
    all services containerized
    docker-compose for full stack

Phase 8: CI/CD
    DevOps project requirements tied to this app
    public git repository
    dockerize services
    docker-compose orchestration with app + database
    CI platform: GitHub Actions / GitLab CI / Jenkins / similar
    pipeline on git push builds and publishes Docker images to registry
    optional CD bonus: deploy to server/cloud with Docker/Kubernetes/Argo CD/similar
    Kubernetes:
        app Deployment with ConfigMaps/Secrets
        app Service
        app Ingress
        database StatefulSet with ConfigMaps/Secrets
        separate namespace on cluster + working demo

    README
